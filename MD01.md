- 제목
    
    ```
    # 을 이용해 제목 작성 가능 #(h1)~#####(h5)
    ```

    ## 예시

- 목록
    ```
    들여쓰기를 통해 목록 생성 가능
    1. 과일
        1. 사과
        2. 귤
        3. 딸기
    - 동물
        - 개
        - 고양이
    ```

- 링크
    
    ```
    [표시될 글](링크경로) 로 링크 작성 가능
    [구글](https://google.com)
    [파이썬 코드](./main.py)
    ```

    [구글](https://google.com) 

    [파이썬 코드](./main.py)

- 이미지

    ```
    ![설명글](링크경로) 로 이미지 업로드 가능
    ![예시](./pocket01.png)
    ```

    ![예시](./pocket01.png)

- 테이블
    ```
    |내용| 으로 하나의 칸을 만들고 연결해서 테이블 생성 가능

    |이름|나이|

    |홍길동|60|

    ```
    *텍스트* : 글 기울임

    ~~텍스트~~ : 글 삭선
    --- : 가로줄 생성
